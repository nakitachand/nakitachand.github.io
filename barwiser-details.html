Rebar Cutlist Generator (Barwiser)

VBA Tool generating optimized cutlists from engineered cage designs

Summary

The objective of this tool was to generate a production ready cutlist for rebar of any required size and quantity based on engineered cage designs. The VBA tool extrapolates table data pasted from a pdf file, runs the required resource calculations, and outputs a cutlist optimized for both minimized waste and operator machine entry. Ultimately the future goal of this tool was to directly implement the logic from an office computer and send the required cutlists remotely to reduce human error and reduce production costs.
